# Site Testimonials Marquee Design

## Goal

Replace the static testimonial masonry with a cleaner, more dynamic review wall.

## Direction

- Use two horizontal marquee rows.
- Run the rows in opposite directions to create motion without visual noise.
- Keep the cards compact and consistent in height.

## Rules

- Preserve the existing testimonial copy.
- Improve hierarchy with stronger role labels and more structured author blocks.
- Slow down motion and reduce it further on hover for readability.
- Keep mobile readable by using narrower cards and stacked rows.
