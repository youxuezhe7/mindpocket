- **Browser-based CLI sign-in** with device approval makes it much easier to connect local tools to your MindPocket account.
- **Agent-ready CLI workflows** now cover more everyday operations, making automation and scripting practical instead of experimental.
- **Public docs with search** gives new users a clearer path to setup, deployment, and ingestion architecture.
- **Clearer extension controls** reduce friction when switching capture behavior and checking sign-in state.
- **Higher reliability across auth and API flows** improves confidence when using MindPocket from scripts, tests, and multi-step workflows.
